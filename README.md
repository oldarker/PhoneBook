Телефонная книга

Программа для управления контактами

Функции
- Добавление, удаление, изменение контактов
- Проверка на дубликаты (имя и номер)

 Технологии
- Java
- Swing
- MS SQL Server + JDBC

Как запустить
1. Скопируйте и переименуйте `Configexample.java` → `Config.java`
2. Подставьте свои данные для подключения к БД
3. Откройте и запустите проект
